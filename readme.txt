Vamos as explicações sobre as tags utilizadas na estrutura projeto:
 Iniciamos com uma estrutura em HTML5 onde:

- Tag <!DOCTYPE HTML> indica que temos uma linguagem em HTML 5.

- Tag <html lang> indica o idioma do site.

- Tag <head> área utilizada para definir as compatibilidades tanto com navegadores e formas
de linguagem, e também onde são definidas as opções gráficas do projeto.

- Tag <body> área utilizada para desenvolver o projeto em HTML.

 Tags utilizadas dentro da áre da tag <boddy>:

- Tag <h1> utilizada para defiinir texo como título em destaque.

- Tag <form method="post"> utilizado para abrir um fomrulário onde são preenchidos informações
pedidas ao usuário.

- Tag <div> utilizada para abrir campos separados.

- Tag <label for=""> utilizada para receber os dados em uma variável.

- Tag <input type> utilizada para definir o tipo da variável.

- Tag <textarea> utilizada para abrir uma caixa de texto.

- Tag <select> utlizada para abrir uma caixa de opções.

- Tag <option> utilizada para definir as categorias dentro da tag <select>.

- Tag <button> utlizada para realizar o submit dos dados preenchidos.
